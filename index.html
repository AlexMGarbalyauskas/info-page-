Technology Stack

Built with modern, industry-standard technologies for performance, security, and scalability

Full-Stack Architecture

A complete full-stack social fitness web application combining workout tracking, social networking, and real-time communication features.

Frontend

React v19.2.1

Modern Single Page Application (SPA) with React Router, Context API, and hooks. Features include real-time feeds, messaging, and social interactions.

Backend

Node.js + Express v5

RESTful API backend handling authentication, file uploads, database operations, and WebSocket connections for real-time features.

Database

SQLite Database

ACID-compliant relational database with migration system. Manages users, posts, comments, messages, and workout schedules efficiently.

Real-Time

Socket.IO v4

Real-time bidirectional communication for instant messaging, notifications, message reactions, and live updates across the platform.

Styling

Tailwind CSS v3

Utility-first CSS framework with custom theme, dark mode support, responsive design, and consistent styling across all components.

Security

Security & Auth

JWT authentication, bcrypt password hashing, Google OAuth 2.0, end-to-end message encryption (AES-GCM), and Helmet security headers.

Core Technologies & Libraries

React 19 Node.js Express 5 SQLite3 Socket.IO JWT Passport.js Multer Axios Tailwind CSS Web Crypto API bcrypt

Deployment & DevOps

Hosted on Render.com cloud platform with automatic deployment, environment configuration, database persistence, and CDN-backed static assets.

About the Project

Project Purpose & Vision

Stay Fit is a comprehensive final year project that tackles the modern challenge of maintaining consistent fitness habits in our busy digital lives. The application was created to demonstrate full-stack development expertise while solving a real-world problem: making fitness accessible, social, and enjoyable.

Traditional fitness apps often fail to maintain user engagement due to lack of social elements and overwhelming complexity. Stay Fit addresses this by combining short workout challenges, social networking features, and real-time communication into a unified platform that makes staying active feel natural and fun.

How It Was Built

This project showcases modern software engineering practices through:

  • Full-Stack Development: React frontend communicating with Node.js/Express REST API
  • Database Design: Normalized SQLite schema with migration system for version control
  • Real-Time Features: WebSocket implementation using Socket.IO for instant messaging and notifications
  • Security Implementation: Multi-factor authentication (JWT + OAuth), password hashing, and end-to-end encryption
  • Cloud Deployment: Professional hosting on Render.com with CI/CD pipeline
  • Responsive Design: Mobile-first approach with Tailwind CSS and PWA capabilities
